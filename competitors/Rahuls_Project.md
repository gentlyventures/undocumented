# Competitor Profile: Rahul's Project (Apprentice Systems)

## 1. Project Information
- **Testing Access**: [apprentice-systems.com](https://apprentice-systems.com/)
- **Description**: AI-native Credit Appraisal automation tool for banking relationship managers and credit analysts.

---

## 2. Problem Statement
- Bank relationship managers and credit analysts waste hours compiling massive portfolios of raw financial customer data into credit memos or executive presentations.

---

## 3. Technical Architecture & Google Cloud Integration
- **Features**:
  - Ambient Agent execution: runs asynchronously in the background.
  - Verification & Trust: provides inline citations back to source client documentation.
  - Tuned evaluation harness: utilizes specialized testing patterns to optimize accuracy.
- **Platform**: Web platform.

---

## 4. Business Case & ROI
- Speeds up manual credit appraisal review processes.
- Eliminates grunt work by auto-drafting first-pass memos.
- Promotes confidence via verifiable inline source citations.
