## 2. Problem Statement
- Queensland construction practitioners must navigate 35+ fragmented government procurement, contractor qualification, and licensing data feeds.
- Schema shifts and tender format updates cause silent system failures, leaving practitioners blind to pipeline developments.

---

## 3. Technical Architecture & Google Cloud Integration
- **Features**:
  - Substrate Brain: BigQuery warehouse consolidating 20.1M-record Australian Business Number registry, address mapping, license records, and historical council records.
  - AI Council: Multi-model orchestration (Claude for reasoning/MCP, Gemini Pro for drafting, Gemini Flash for verification/grounding).
  - Iterative loops: Writer/Grader Editor-to-Brief loops catch framing errors and force self-correcting prompt iterations before publishing.
  - Hardened MCP Layer: OAuth-protected MCP server running on Compute Engine with 14 tools (BigQuery scans, database sync, human-in-the-loop write gates).
  - Web dashboards: live pipeline metrics, workforce registers, and tender updates at `qld2032.com`.
  - Telemetry & Hardening: drift checkers, freshness cron assertions, and cost-disciplined invalidation caches.

---

## 4. Business Case & ROI
- Consolidates 35+ fragmented data portals into a real-time intelligence hub.
- Prevents database sync silence/failures.
- Enables tier-1 builders and subcontractors to track upcoming tenders and compliance requirements.
