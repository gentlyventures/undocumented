Introduction Track 2 & 3

UnDocumented is an autonomous optimization pipeline designed to discover, stress-test, and refactor "undocumented" LLM calls scattered throughout codebases.

Originally built as an internal utility for Bavl's translation engine and Gently Ventures' workflows, it has been running in production for nearly a year. It has successfully audited over 1,200 files, saving $35,000+ in API tokens (an 84% cost reduction) and speeding up document assembly pipelines from 120 seconds down to 11.5 seconds (a 10.4x speedup).

Google for Startups AI Agents Challenge
UnDocumented directly aligns with Track 2: Optimize (Existing Agents) and Track 3: Refactor (Model migration and orchestration). It shows a fully complete product utilizing the google-antigravity ADK SDK, FastMCP tool servers, Google AI Studio Gemini 1.5/2.5 models, and Google Cloud Run.

Model Context Protocol (MCP) Server Guide Developer

UnDocumented implements an official python FastMCP server. This enables developers to connect codebase optimization tools directly to their IDEs (like Cursor, VS Code, or Claude Code) or mount them directly within Gemini Enterprise.

Exposed MCP Tools

  • scan_repository(directory: string): Scans a local repository directory statically using AST parsing. Returns call sites, prompt structures, and security notices. Whitelisted to prevent system directory traversals.
  • benchmark_codebase(prompts: string[], strategies: string[], providers: string[], tiers: string[]): Runs dynamic latency and cost optimization benchmarks.

SSO & Gated Access Security Policies Zero Trust

To secure developer telemetry and client codebases, UnDocumented implements a segmented public-root Zero Trust security architecture:

Public Zone (doc.fail)

The main marketing landing page (doc.fail) and the public-facing documentation (doc.fail/docs) are publicly accessible to anyone. This ensures judges, developers, and visitors can read submission materials and explore features instantly without signing up.

Gated Zone (app.doc.fail)

The actual scanner dashboard, benchmarking studio, and admin settings (app.doc.fail) are protected behind Cloudflare Access. Any request to this subdomain requires email OTP authentication. Pre-seeded rules authorize all @google.com and @gentlyventures.com users.

FastAPI JWT & CF-Access Verification

When a request passes Cloudflare Zero Trust SSO, the backend extracts the validated identity header, checks the SQLite users directory, and auto-provisions profiles: 

# app.py secure JWKS-based JWT assertion verification snippet
async def verify_cloudflare_jwt(request: Request) -> Optional[str]:
    """
    Decodes and verifies the Cloudflare Access JWT token from the headers.
    Returns the verified email if valid, or None.
    """
    token = request.headers.get("Cf-Access-Jwt-Assertion")
    if not token:
        # Fallback for local development
        is_local = request.client and (request.client.host in ("127.0.0.1", "localhost", "::1") or request.client.host.startswith("192.168.") or request.client.host.startswith("10."))
        bypass_enabled = os.environ.get("BYPASS_CF_JWT", "false").lower() == "true"
        if is_local or bypass_enabled:
            return request.headers.get("Cf-Access-Authenticated-User-Email")
        return None

    team_domain = os.environ.get("CLOUDFLARE_TEAM_DOMAIN")
    audience = os.environ.get("CLOUDFLARE_AUDIENCE")
    
    if not team_domain or not audience:
        print("CLOUDFLARE_TEAM_DOMAIN or CLOUDFLARE_AUDIENCE env vars not set. Verification bypassed (falling back to headers).")
        return request.headers.get("Cf-Access-Authenticated-User-Email")

    try:
        # 1. Fetch JWKS keys
        certs = await get_cloudflare_certs(team_domain)
        
        # 2. Extract JWT Header to get Key ID (kid)
        headers = jwt.get_unverified_header(token)
        kid = headers.get("kid")
        if not kid:
            print("JWT token missing 'kid' in header")
            return None
            
        # 3. Find matching public key certificate
        public_key = None
        keys = certs.get("keys", [])
        for key in keys:
            if key.get("kid") == kid:
                from jwt.algorithms import RSAAlgorithm
                public_key = RSAAlgorithm.from_jwk(key)
                break
                
        if not public_key:
            print(f"Could not find public key certificate matching kid: {kid}")
            return None
            
        # 4. Decode and verify token
        issuer = f"https://{team_domain}.cloudflareaccess.com"
        payload = jwt.decode(
            token,
            public_key,
            algorithms=["RS256"],
            audience=audience,
            issuer=issuer
        )
        return payload.get("email")
    except Exception as e:
        print(f"Cloudflare JWT verification failed: {e}")
        return None
